Getting Started

An overview of Ambient, how to download and use, basic templates and examples.

Ambient makes use of HTML section elements and CSS properties to appropriately scale as the device or view port size increases. We also include an optional library of 4 distinct, attribute based design patterns for layout.

Ambient has a core fluid design system for HTML sections as a presentation format without the need for structure based classes.

Ambient is about HTML as a presentation format. Design patterns are about defining namespaces for the HTML to live in. It works by generating styles for elements, attributes & values. And it's the key to unlocking this new approach.

Ambient is downloadable in two forms, and include the following directories and files:

Ambient

└── css/
    ├── lib/

This is the most basic form of Ambient for a foundation in any web project. The css/ folder includes the core fluid design system CSS. The lib/ folder includes the design patterns.

am·bi·ent /'ambēənt/ adjective

  1. of the surrounding area or environment
  2. completely surrounding; encompassing
  3. creating a certain reaction or mood, often a subconscious one, by being wherever people tend to be
  4. pertaining to or noting sounds that create a peaceful and relaxed atmosphere
  5. pertaining to or noting close and constant social contact and communication fostered by the Internet or the use of digital devices
