Introduction

Ambient is a CSS framework to layout the document outline

A lightweight and modular HTML5 UI development methodology

Boba Fett release v1.3.3

Bower 


$ bower install ambient

Install Ambient package. 


ambient/
├── dist/
│    └── css/
│        ├── ambient.css
│        └── ambient.min.css
└── less/
      ├── core/
      ├── mixins/
      └── ambient.less

The Ambient package includes both the dist/ and less/ files.

Optional 


$ bower install ambient#dist --save

Install only the latest version dist/ CSS files and save Ambient as a dependency in your project.

Basic HTML5 example

Build the intent of your content. You can add additional am-layout values as well as extend child attributes am-SOMETHING to your scaffolding to gradually build a hierarchy that defines your layout.

See the Pen Idea for flexbox HTML5 scaffolding & layout attributes by I only see spots (@ionlyseespots) on CodePen.

Code

Start with this basic HTML5 example. 


<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="utf-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<title>Ambient starting HTML5 example</title>
<meta name="description" content="">
<meta name="viewport" content="width=device-width, initial-scale=1">

<link rel="apple-touch-icon" href="apple-touch-icon.png" >
<!-- Place favicon.ico in the root directory -->
<!-- Ambient -->
<link rel="stylesheet" href="ambient/css/ambient.min.css">
</head>
<body>
<header role="banner">
   <h1>
        header
   </h1>
</header>
<aside>
    <h1>
        aside
    </h1>
</aside>
<main role="main">
    <h1>
        Hello, world!
    </h1>
</main>
<aside>
    <h1>
        aside
    </h1>
</aside>
<footer role="contentinfo">
    <h1>
       footer
    </h1>
</footer>
</body>
</html>

Ambient is a lightweight and modular HTML5 UI development methodology.

This system evolved out of the need to fully decouple markup (HTML) from presentation (CSS).

Ambient makes use of HTML5 elements & CSS3 attributes to layout the document outline.
