Download

Install with Bower 

$ bower install ambient

This is the most basic form of Ambient. 

ambient/
├── css/
│   ├── ambient.css
│   └── design-patterns.css
└── js/
     └── prefixfree.js

The css/ folder includes the base grid system and optional design patterns.

The js/ folder includes a client side vendor prefixer that adds the current browser’s prefix to any CSS code, only when it’s needed.

Ambient

am·bi·ent /'ambēənt/ adjective

  1. of the surrounding area or environment
  2. completely surrounding; encompassing
  3. creating a certain reaction or mood, often a subconscious one, by being wherever people tend to be
  4. pertaining to or noting sounds that create a peaceful and relaxed atmosphere
  5. pertaining to or noting close and constant social contact and communication fostered by the Internet or the use of digital devices

Starter template

Start with this starter HTML5 template, or modify our examples.

Copy the starter HTML5 template below to begin a web project with Ambient, or modify our examples. 

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="utf-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<title>Starter HTML template</title>
<!-- Ambient -->
<link href="ambient/css/ambient.css" rel="stylesheet">
<link href="ambient/css/design-patterns.css" rel="stylesheet">
<script src="ambient/js/prefixfree.js"></script>
</head>
<body>
<header role="banner">
   <h1>Hello, header!</h1>
</header>
<hr />

<nav role="navigation">
    <a href="#">nav</a>
    <a href="#">nav</a>
    <a href="#">nav</a>
</nav>
<hr />

<main role="main">
    <article role="row">
            <h1>Hello, world!</h1>
    </article>
</main>
<hr />

<footer role="contentinfo">
       <h1>Hello, sticky footer!</h1>
</footer>
</body>
</html>

Ambient is a methodology for creating layout.

This system evolved out of the need to fully decouple markup (HTML) from presentation (CSS). To do that, I believe markup needs to become the presentation. Or more specifically, presentation format.

Our goal is to deliver a lean, lightweight base grid system for HTML to convey presentation format (layout), while also maintaining a separation from presentation style (design) that adapts across different devices.

We also extend HTML’s elements, attributes and values into design patterns to help you display your application’s UI clearly.

Ambient is about HTML as a presentation format. Design patterns are about defining namespaces for the HTML to live in. It works by generating styles for elements, attributes & values. And it's the key to unlocking this new approach.
