Getting started

Introduction

Ambient is a method for creating layout not constrained by the browser

HTML5 sectioning elements within Ambient define themselves, creating a visual structure for the document outline. This means you can build content into simple or complex layouts.

Download

Bower: v1.2.3 Source: v1.2.4

Bower 


$ bower install ambient

This is the basic form of Ambient. 


ambient/
└── css/
    ├── ambient.css
    └── ambient.min.css

The css/ folder includes the semantic layout and UI patterns. The file comes as a single CSS file as well as a minified version.

Alternatively, get the ambient.min.css file from the repo and save the file in your directory.

Source
Clone, fork or download the repo for complete source code, docs and this website. Ambient source files are in the less/ folder. Source code is the most current development (unstable) version of Ambient.

Ambient is ready to adopt whatever your design or style is

Back to basics. Ambient is a lightweight and modular HTML5 UI development methodology. Feel free to use semantic HTML5 tags, whenever and wherever you like.

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="utf-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">

</head>
<body>
<header role="banner">
   <h1>
        Header
   </h1>
</header>
<hr />

<nav role="navigation">
    <a href="#">nav</a>
    <a href="#">nav</a>

</nav>
<hr />

<main role="main">
    <article>
        <section>
            <h1>
                Hello, world!
            </h1>
        </section>
        <aside>
            <h1>

            </h1>
        </aside>

<footer role="contentinfo">

Ambient is a methodology for creating layout

This system evolved out of the need to fully decouple markup (HTML) from presentation (CSS). While presentation changes can require changes in structure, content can be changed without any need for structural change beyond the semantic layout. Our goal is to deliver semantic layout for HTML elements, attributes and values, while also maintaining a separation from content that scales up from Portrait to Landscape.

Ambient makes use of HTML5 elements & CSS3 properties to layout semantic markup. Ambient UI patterns are layout-related attributes for content.
