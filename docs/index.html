Getting Started

An overview of Ambient, how to download and use, basic templates and examples.

Download

Ambient is downloadable in two forms, and include the following directories and files:

Ambient 

ambient/
└── css/
    ├── lib/
    ├──── navs.css
    ├──── ux-shells.css
    └── ambient.css

This is the most basic form of Ambient for a foundation in any web project. The css/ folder includes the core fluid design system for our CSS. The lib/ folder includes optional design patterns.

Ambient Design

am·bi·ent /'ambēənt/ adjective

  1. of the surrounding area or environment
  2. completely surrounding; encompassing
  3. creating a certain reaction or mood, often a subconscious one, by being wherever people tend to be
  4. pertaining to or noting sounds that create a peaceful and relaxed atmosphere
  5. pertaining to or noting close and constant social contact and communication fostered by the Internet or the use of digital devices

Basic template

Start with this basic HTML5 template, or modify our examples.

Copy the HTML below to begin a project with Ambient. 

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="utf-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<title>Basic HTML template</title>

<!-- Ambient -->
<link href="ambient/css/ambient.css" rel="stylesheet">
<link href="ambient/css/lib/ux-shells.css" rel="stylesheet">
</head>
<body>
<header role="banner">
   <h1>Hello, header!</h1>
</header>
<hr />

<nav role="navigation">
    <a href="#">nav</a>
    <a href="#">nav</a>
    <a href="#">nav</a>
</nav>
<hr />

<main role="main">
    <article role="row">
            <h1>Hello, world!</h1>
    </article>
</main>
<hr />

<footer role="contentinfo">
       <h1>Hello, sticky footer!</h1>
</footer>
</body>
</html>
