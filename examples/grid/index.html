Alternatively, a single <section> also displays full width.

        <p>

    </section>
</article>

Orientation: Portrait and Landscape.

Two default orientations, Portrait & Landscape, make Ambient ideal for responsive layouts.

Column clearing

Traditional HTML & CSS column/grid clearing makes use of article, header, nav base elements as well as 100% width elements h1, h2, h3, h4, h5, h6, p, footer, hr.

Et cetera

Ambient makes use of HTML5 elements & CSS3 properties to convey a consistent visual structure within a semantic grid.

Design patterns are about defining namespaces for the HTML to live in. It works by generating styles for elements, attributes & values. And it's the key to unlocking this new approach.
