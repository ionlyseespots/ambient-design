Full width, single column

An <article> displays full width by default and defines self-contained content that could exist independently of the rest of the content.

Alternatively, a single <section> also displays full width by default and defines a section in the document.

    <section>
        <p>

    </section>
</article>

Orientation: Portrait and Landscape.

Two default orientations, Portrait & Landscape, make Ambient ideal for responsive layouts.

Column clearing

Clearing a <section> makes use of hr as well as header, footer, nav, h1, h2, h3, h4, h5, h6, p elements.

<hr> represents a thematic break between paragraph-level elements (for example, a change of scene in a story, or a shift of topic with a section).

<header> defines the header of a page or section.

<footer> defines the footer for a page or section.

<nav> defines a section that contains only navigation links.

<h1, h2, h3, h4, h5 ,h6> briefly describes the topic of the section.

<p> defines a paragraph of text and is left justified.

Ambient makes use of HTML5 elements & CSS3 properties to convey structure within a semantic grid.

Design patterns are about defining namespaces for the HTML to live in. It works by generating styles for elements, attributes & values. And it's the key to unlocking this new approach.
