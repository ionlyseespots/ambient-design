Ambient grid examples

Basic grid layouts to get you familiar with visualizing HTML as a presentation format within the Ambient base grid system.

Three equal columns

Get three equal-width columns starting at orientation portrait and scaling to orientation landscape. On orientation portrait, the columns will automatically stack.

Remember, equal grid columns default to flex: 1;. Equal columns make use of the section element and defines a section in a document.
