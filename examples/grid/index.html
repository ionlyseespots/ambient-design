Basic page layout

Ambient is a method for creating semantic layout not constrained by the browser.

DO use HTML5 elements in Ambient create to a semantic layout for your content.

Do NOT use HTML5 elements as a replacement for divs or classes.

Full width single column

An <article> displays full width.

Alternatively, a single <section> also displays full width. 


<article>
    <section>
        <p>
            Alternatively, a single <section> also displays full width.
        </p>
    </section>
</article>

Complex nested equal columns

Nesting gives you columns starting at orientation portrait and scaling to orientation landscape, within the larger column. On orientation portrait, the columns will automatically stack.

Orientation: Portrait and Landscape.

Two default orientations, Portrait & Landscape, make Ambient ideal for responsive layouts.

Column clearing

Traditional HTML & CSS column grid clearing makes use of article, header, nav base elements as well as 100% width elements h1, h2, h3, h4, h5, h6, p, footer, hr.

Et cetera

Ambient makes use of HTML5 elements & CSS3 properties to convey a consistent layout.

UI patterns are about defining namespaces for the HTML to live in. It works by generating styles for elements, attributes & values. And it's the key to unlocking this new approach.
