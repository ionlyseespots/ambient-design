Semantic

Ambient makes use of HTML5 sections & outlines for semantic scaffolding.

Layout

Ambient makes use of CSS3 attributes on elements for page layout.

Design

Ambient does not style components by default.

Because we have removed the dependency on base styles, you do not need to override Ambient core CSS to customize the design. Once your semantic layout is in place, you can then customize the design with optional Ambient skin classes, or your own custom classes.
