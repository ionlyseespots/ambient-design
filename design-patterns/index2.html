Ambient: An advanced CSS framework with a difference

Ambient is an intuitive front-end framework for fast, app website development. It avoids presentational CSS classes in the HTML markup itself, instead using LESS to control the layout in the stylesheet.

Ambient differs from traditional CSS frameworks in that it provides a solid foundation for HTML5 layout.

Ambient extends the power and functionality of CSS3 to HTML5 elements and includes a grid, making it more than enough to get started. And it's completely semantic and responsive, meaning the layout adapts to the orientation of the user's screen or device, using media queries.

Some practical benefits of this approach are:

  • Separation of markup and presentation
  • No presentational CSS classes in the HTML markup itself
  • Call mixins (functions) from your CSS layout attributes
  • Nest CSS layout attributes
  • Customizable design patterns
