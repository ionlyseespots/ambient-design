Layout

Ambient avoids presentational CSS classes in the HTML markup.

Do use namespaces to define your layout.

Do NOT use namespaces as a replacement for styling content better suited to a class name.

Namespaces

Because we define our own custom attributes, we can define properties to all HTML elements.

Global

Global attributes are layout properties common to all HTML elements.

am-margin
am-padding
am-layer

Define your layout

The am-layout attribute must be used on a root semantic element, though the attribute may have no effect on some elements.

Global

Global layout attributes are layout properties common to all HTML elements.

Whitespace

Ambient does not style margin & padding by default.

Margin

Add the attribute am-margin to any element to add space outside an element.

If needed, additional size values large & small are available.

Padding

Add the attribute am-padding to any element to add space inside an element.

If needed, additional size values large & small are available.

Layers

Add the attribute am-layer="VALUE" to any element to add a layer depth value (z-index).

If needed, the additional skin classes for shadows .am-shadow-VALUE are available. *Shadows can be used on any element.

Grid layout

Grid layout makes use of "blocks" to position content, and thus relies on the root semantic element for intent.

Blocks

Add the layout attribute am-block to a div element to create a grid layout. Blocks are infinitely nestable.

Add the width value am-block="VALUE" to create widths and off-set widths.

Add the letter P and/or L to the width value to create unique Portrait and/or Landscape widths and off-set widths.

Feel free to mix & match orientations and width values.

Code 

<section>
    <div am-block>
            1
    </div>
    <div am-block>
        <div am-block="6">
            1/2
        </div>
        <div am-block="6">
            2/2
        </div>
    </div>
    <div am-block>
        <div am-block="4">
            1/3
        </div>
        <div am-block="4">
            1/3
        </div>
        <div am-block="4">
            1/3
        </div>
    </div>
    <div am-block>
        <div am-block="3">
            1/4
        </div>
        <div am-block="3">
            1/4
        </div>
        <div am-block="3">
            1/4
        </div>

Forms

Form layout is used to structure forms, and thus relies on the root semantic fieldset element for intent.

Inline form

<fieldset am-layout="inline">

Code 


                        <form>
                        <fieldset am-layout="inline">
                        <div role="group">
                        <legend>
                        My form
                        </legend>
                        <label for="inline-email">Email</label>
                        <input id="inline-email" type="email" placeholder="Email">

                        <label for="inline-password">Password</label>
                        <input id="inline-password" type="password" placeholder="Password">

                        <label for="inline-remember">
                        <input id="inline-remember" type="checkbox"> Remember me
                        </label>

                        <button type="submit" class="am-btn am-primary">Sign in</button>
                        <a href="#">Forgot your password?</a>
                        <div>
                        </fieldset>
                        </form>
