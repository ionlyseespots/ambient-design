Design Patterns

Design patterns are built on Ambient's core fluid design system.

Ambient design is methodology for creating layout. There are 5 distinct design patterns for layout:

  1. Typography
  2. UI Elements
  3. UX Shells (first priority)
  4. Navs
  5. Form States (second priority)
