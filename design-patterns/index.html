Layout

Introduction

Ambient is page layout

Ambient displays page layout for defined content.

We define our own namespaces

Because we define our own custom attributes, we can define extendable properties common to all HTML elements as well as HTML5 semantic elements.

Do use CSS3 attributes to layout defined content.
Do NOT use CSS3 attributes as a replacement for styling content better suited to a class name.

Global and Root namespaces

Global attributes

Global attributes are layout properties common to all HTML elements. These attributes can be used on any element.

am-margin
am-padding
am-layer

Root attribute

Define your layout. The root layout attribute must be used on a root HTML5 semantic element, though the attribute may have no effect on some elements. Once you've defined the root element, you can add additional layout properties to your scaffold as child attributes to gradually build a hierarchy that defines your layout.

am-layout

Global attributes

Global attributes are layout properties common to all HTML elements. These attributes can be used on any element.

Margin

Default margin is 0
To add space outside an element, add the attribute am-margin to any element.

If needed, specific direction or size is available as:

  • am-margin="top"
  • am-margin="right"
  • am-margin="bottom"
  • am-margin="left"
  • am-margin="small"
  • am-margin="large"

Padding

Default padding is 0
To add space inside an element, add the attribute am-padding to any element.

If needed, specific direction or size is available as:

  • am-padding="top"
  • am-padding="right"
  • am-padding="bottom"
  • am-padding="left"
  • am-padding="small"
  • am-padding="large"

Layers

Default layer is 0
To add depth to an element, add the attribute am-layer="DEPTH VALUE" to any element.

Layer depths are available as

  • am-layer="z-1"
  • am-layer="z-2"
  • am-layer="z-3"
  • am-layer="z-4"
  • am-layer="z-5"

Grid layout

Grid layout is used to structure content, and thus relies on the root HTML5 semantic element for intent.

Basic grid layout with equal widths

To create a basic grid add the attribute and value am-layout="grid" to any HTML5 semantic element. On orientation portrait, grid layout does NOT automatically stack.

Basic

To add a row, add the attribute and value am-layout="row" to a child div.
To add a column, add the attribute and value am-layout="column" to a child div.

<section am-layout="grid">

Ambient is a lightweight and modular HTML5 UI development methodology.

This system evolved out of the need to fully decouple markup (HTML) from presentation (CSS).

Ambient makes use of HTML5 elements & CSS3 properties to layout the document outline.
