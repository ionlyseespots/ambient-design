Compatibility

Ambient is designed for and tested on numerous browsers and devices. It's built to layout HTML5 and makes use of the CSS3 Flexible Box (flexbox) Layout Module.
Here's what to expect for compatibility.

Ambient

Ambient is a framework you can use to build flexible, clean code for app websites that work in portrait or landscape orientation.

I've taken what I've learned from working with traditional CSS frameworks to build an entirely new framework for building with HTML5.
