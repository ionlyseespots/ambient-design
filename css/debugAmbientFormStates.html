Form States

Inline Form

To create an inline form with the default form state design pattern, add the attribute ambientformstate to any form.

<form ambientformstate>

Inline Form

Stacked Form

To create a default inline form, add the attribute ambientformstate = the value stacked to any form.

<form ambientformstate="stacked">

Stacked Form

Aligned Form

To create an aligned form add the attribute ambientformstate = the value aligned to any form.

In an aligned form, the labels are right-aligned against the form input controls, but on portrait revert to a stacked form.

<form ambientformstate="aligned">

Ambient Design

Multi-column Form

To create a multi-column form add the attribute ambientformstate = the value multicolumn to any form.

<form ambientformstate="multicolumn">
