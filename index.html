Ambient is a semantic grid system.

Ambient is a methodology for creating layout with 4 distinct design patterns.

Features

Ambient is a semantic grid system for HTML elements, attributes and values.
which is simple to use, extendable and easy to customize.

LESS

Built with LESS to write well-structured, extendable code which is easy to maintain.

Flexbox

A very basic flexbox grid and style with a clean separation of content and presentation.

Patterns

A collection of design patterns using consistent and conflict-free namespaces.

Responsive

Two default orientations, Portraint & Landscape, make Ambient ideal for responsive layouts.

Ambient is a methodology for creating layout.

Ambient makes use of HTML5 elements & CSS3 properties to convey structure within a semantic grid.
While presentation changes can require changes in structure, content can be changed without any need for structural change beyond the semantic grid.

The semantic grid avoids presentational CSS classes in the HTML markup itself.

Open Source

Our goal is to deliver a lean, lightweight semantic grid system for HTML elements, attributes and values
while also maintaining a separation from content that adapts across different devices.
