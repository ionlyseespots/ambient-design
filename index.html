Features

Ambient is a CSS framework built as a lightweight & modular HTML5 layout methodology. Think of it as content aware CSS.

Semantic

Ambient makes it easy for you to layout HTML.

Layout

Avoids presentational CSS classes in the HTML markup.

Orientation

Provides layout for both Portrait & Landscape.

Design

Design is pure, minimal and transparent. Ambient does not style components by default.

Sometimes I have a UI that plays well with a traditional grid paradigm but sometimes I need a more complex grid. This can mean learning grid classes for structure with floats, columns and media queries. I liked reusing classes for design but I became frustrated with classes for structure that were not very clear.

I wanted to build a lightweight and modular HTML5 layout methodology. The CSS should make use of HTML5 (no classes), and provide layout for both portrait & landscape orientations.
