Introduction

Ambient is semantic

Ambient displays semantic scaffolding for HTML5 sections & outlines.

Scaffolding

On orientation portrait, explicit sections will automatically stack.

Do use HTML5 sectioning elements to scaffold content in the document outline.
Do NOT use HTML5 sectioning elements as a replacement for content better suited to a div.

Build the intent of your content

Back to basics. Feel free to use HTML5 semantic elements to scaffold content in the document outline, whenever and wherever you like.

You can add additional layout properties to your scaffold as child attributes to gradually build a hierarchy that defines your layout.

Explicit

All content lying inside the body element is part of a section. Sections in HTML5 can be nested. Beside the main section, defined by the body element, section limits are defined either explicitly or implicitly. Explicitly-defined sections are:

HTML5 Scaffold Description
body column Represents the content of an HTML document.
header row Defines the header for a page or section.
nav row Defines navigation links.
article row Represents self-contained content that could be distributed outside the context of the page. Defines content sections in a single direction, vertically or horizontally. You can specify the layout direction with the am-layout attribute. The default orientation is horizontal.
section column Represents a content section.
aside column Represents a content section tangentially related to the content surrounding it.
footer row Defines the footer for a page or section.

Implicit

The HTML Headings Elements (h1 to h6) defines a new, implicit, section when they aren't the first heading of their parent, explicit, sections. The way this implicit section is positioned in the outline is defined by its relative rank with the previous heading in their parent section. If it is of a lower rank than the previous heading, it opens a implicit sub-section of the section.

HTML5 Scaffold Description
<h1> <h2> <h3> <h4> <h5> <h6> Row Briefly describes the topic of the section it introduces.

Ambient is a lightweight and modular HTML5 UI development methodology.

This system evolved out of the need to fully decouple markup (HTML) from presentation (CSS).

Ambient makes use of HTML5 elements & CSS3 properties to layout the document outline.
