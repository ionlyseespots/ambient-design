Form States

Inline Form

To create an inline form with the default form state design pattern, add the attribute am-formstate to any form.

Inline

<form am-formstate>

Stacked Form

To create a default inline form, add the attribute am-formstate = stacked to any form.

Stacked

<form am-formstate="stacked">

Aligned Form

To create an aligned form add the attribute am-formstate = aligned to any form.

Aligned

<form am-formstate="aligned">

Multi-column Form

To create a multi-column form add the attribute am-formstate = multicolumn to any form.

Multi-column

<form am-formstate="multicolumn">

Mix & match form states

To mix & match form states add the attribute am-formstate = value for the design patterns you want to mix & match to any form.

Stacked + Aligned

<form am-formstate="stacked aligned">

Multi-column + Stacked

<form am-formstate="multicolumn stacked ">

Multi-column + Aligned

<form am-formstate="multicolumn aligned ">
